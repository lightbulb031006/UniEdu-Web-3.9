# 🎓 UniEdu 4.0 - Education Management System

Hệ thống quản lý giáo dục hiện đại với kiến trúc Backend/Frontend tách biệt.

## 🏗️ Kiến trúc

```
uniedu/
├── backend/          # Backend API (Node.js + Express + TypeScript)
├── frontend/         # Frontend (React + TypeScript + Vite)
├── docs/             # Documentation
└── supabase/         # Database migrations
```

## 🚀 Quick Start

### **Backend**

```bash
cd backend
npm install
npm run dev
```

Backend chạy tại: `http://localhost:3001`

### **Frontend**

```bash
cd frontend
npm install
npm run dev
```

Frontend chạy tại: `http://localhost:5173`
